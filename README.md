# Library Management

Book and member management with borrow/return. Uses ArrayList. Persists to library.dat.

**Author:** Bisman Singh <bismanmadaan1@gmail.com>

## Features

- Book: id, title, author, isbn, available
- Member: id, name
- Add/remove books and members
- Borrow and return books

## Build & Run

```bash
make
make run
```

## Menu

1. Add book
2. Remove book
3. Add member
4. Remove member
5. Borrow book
6. Return book
7. List books
8. List members
9. Quit
